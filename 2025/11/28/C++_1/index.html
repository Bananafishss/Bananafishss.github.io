C/C++ 学习日记 基本语法

1.C++ 基本语法

第一个C++程序

‘’’cpp
#include
int main(int argc, char argv)
//char：一个字母
//char：一句话
//char*：一组话
{
int n;
std::cout<<”Input a Entier: “<<std::endl;
std::cin >> n;
for(int i=0; i<n; i++){
std::cout<<”hello world” << std::endl;
}
return 0;
}
‘’’

C++的输入/输出

using namespace std;

  1. std::cout<< “contenu” << std::endl(换行符);
    如:cout<<”<”<< this->x <<”,”<< this->y << “>” <<endl; 输出结果为：<x,y>
  2. std::cin>>（相当于scanf） “input contenu”;
  3. 多个输出可以相连：std::cout << “A = “ << a << “, B = “ << b << std::endl;
  4. 缩写方法：如果每次不想重复写std:: 可以在本文件开头写 using namespace std;
  5. cin 可以输入字符串如：
    ‘’’cpp
    string name;
    cin >> name; // 只能读一段不带空格的
    string line;
    getline(cin, line); // 可以读空格
    ‘’’

C++的命名空间

  1. 命名空间（namespace）相当于一个有独特名字的仓库，可以往里面不停的塞东西：
    ‘’’cpp
    namespace myspace{
    int a = 10;
    void hello() {
    std::cout << “Hi!” << std::endl;
    }
    }
    std::cout << myspace::a;
    myspace::hello();
    ‘’’
    可以在不同文件、不同位置 不断往同一个 namespace 填东西。
    ‘’’cpp
    在a文件中：
    namespace x{
    int a=6;
    }
    在b文件中：
    namespace x{
    int b = 5;
    }
    在c文件中我既可以使用a也可以是使用b。
  2. 匿名namespace
    只能在本文件使用的namespace。
  3. 嵌套的namespace 可以通过创建一个别名来使引用更方便：namespace N = AReallyLongName::AnotherReallyLongName;
    ‘’’cpp
    namespace AReallyLongName {
    namespace AnotherReallyLongName {
    int f();
    int g();
    void bar(int x, int y);
    }
    }
    void foo() {
    namespace N = AReallyLongName::AnotherReallyLongName;
    N::bar(N::f(), N::g());
    }
    ‘’’

C++的引用机制

  1. 类似c中的指针 当定义一个变量a的引用是b后 b就是a的别名，修改b就是修改a本身。
    ‘’’cpp
    int a = 10;
    int &b = a; // b 是 a 的引用（外号）
    b = 20; // 其实就是把 a 改成 20
    std::cout << a; // 输出 20
    ‘’’
  2. 引用的用处
    和c指针的用法一样，用来给函数传递外部的值，告诉函数直接操作变量而不是创建变量的副本。
  3. 和指针的区别
    1. 定义引用：
      int &b = a; b = a = 20;
      定义指针：
      int *b = &a; *b = 20;//b里面是a的地址
    2. 引用的规则：
      1. 引用必须在声明时绑定对象：（int &b = a;对） （int &b;错）
      2. 引用一旦绑定，就不能更换绑定对象，它不像指针那样能指来指去。
      3. 引用永远不为 null（不像指针可能是空的）
      4. 没有嵌套的引用，引用不是在内存中的对象。
  4. 常引用 const &
    形如：
    ‘’’cpp
    void print(const int &x);
    ‘’’
    意思是：
    我不会改你的变量（const）
    但我也不想复制它（引用）
    适用于：string、vector 等大块对象。

内存管理操作符

new就是一个特殊的指针！

  1. 在 C++ 中它们就是两个关键字：
    new（向电脑要内存）
    delete（把内存还给电脑）

  2. 功能与malloc相同，但是用法更简便：
    malloc的用法：麻烦
    int p = (int)malloc(sizeof(int));
    *p = 5; // 你得自己赋值
    new的用法：简单
    int *p = new int(5); // 自动创建 int 并赋值 5

  3. 用途更广：
    首先，new可以用来创建一个动态数组：
    data = new int[tailleMax];//用new新建一个名字为data 元素数据类型为int 元素个数为tailleMax的数组。
    其次，new可以用来创建class的对象：
    Point* p = new Point(1.0, 2.0);
    //等价于调用 Point 的构造函数的同时分配一块能放 Point 的内存。得到的是：一个对象和一个指向对象的指针

  4. delete的用法：
    int *p = new int(5);
    ip2 = new int[10] ;//变量的删除
    delete p ;
    delete [] ip2 ;//数组的删除和内存释放

函数的重载

  1. 在C++中我可以用相同的函数名写不同的函数，只要他们的参数不同（个数，类型，甚至顺序不同即可）。比如：
    ‘’’cpp
    void print(int x) {
    cout << “整数: “ << x << endl;
    }

    void print(double x) {
    cout << “小数: “ << x << endl;
    }

    void print(string s) {
    cout << “字符串: “ << s << endl;
    }
    //调用：
    print(10); // 去找 print(int)
    print(3.14); // 去找 print(double)
    print(“hello”); // 去找 print(string)
    ‘’’

  2. 但是返回值不同不能作为区分的标准！比如：
    ‘’’cpp
    int f(int a);
    double f(int a); // 不行
    ‘’’

  3. 有时调用函数机器会无法识别，比如：
    ‘’’cpp
    void print(double num)
    {
    std::cout << “C’est un double: “ << num << std::endl;
    }
    void print_int(int num)
    {
    std::cout << “C’est un int: “ << num << std::endl;
    }

    print(5);//X错误因为无法识别参数的类型！
    ‘’’

C++中的 const 关键字（常量）

  1. 这是一个前缀可以加载任何量之前，表示这个量无法被修改。对于非指针的变量来说 const的顺序没有影响。
    ‘’’cpp
    void printValue(const int& x) {
    cout << x;
    }//函数参数被const字保护，无法被修改。

    const int a = 261;
    int const a = 261;
    // 没有区别，都是a变量不能被修改
    ‘’’

  2. const 用在指针时，在指针左边就是修饰指针，指针的值不能变，在变量的左边，的右边就是修饰变量，指针的地址不能变。
    ‘’’cpp
    const int     p; // 不能改 p（值不能改）
    int     const p; // 不能改 p（地址不能改）
    const int     const p; // 都不能改
    ‘’’
