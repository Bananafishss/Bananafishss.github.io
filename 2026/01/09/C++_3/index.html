2.C++ 操作符重载

重载的目的

对于一个自定义的类变量（对象）来说，无法直接使用操作符运算，比如对于自定义的 Complexe 类，
‘’’cpp
class Complexe{
private:
double x;
double y;
};
‘’’
对于Complexe a,b; 无法：a+b；所以需要重新定义操作符。

操作符重载的方法

  1. 第一种重载是把操作符重载看作是classe的成员函数（方法）：
    ‘’’cpp
    //A Op B 写作 A.operatorOp(B)
    t1 == t2 // 等价于: t1.operator==(t2)
    t1 += t2 // 等价于: t1.operator+=(t2)
    ‘’’
  2. 第二种重载是把操作符重载当成一个类外的普通函数，然后将被运算的数据作为参数传进操作符函数里：
    ‘’’cpp
    //A Op B 写作 operatorOp(A,B)
    t1 + t2 // 等价于 : operator+(t1,t2)
    t1 * t2 // 等价于 : operator*(t1,t2)
    ‘’’
